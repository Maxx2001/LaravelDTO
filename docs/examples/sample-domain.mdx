## Concept

The Assignments domain handles the creation, management, and lifecycle of work assignments. Users can:

- Create new assignments with specific requirements
- Assign workers to tasks
- Track assignment status and progress
- Complete or cancel assignments

## Belangrijkste Acties

### CreateAssignmentAction

Creates a new assignment in the system.

**Functionaliteit:**
- Validates the input data
- Creates the assignment record
- Sends notification to assignee (if specified)
- Logs the creation event
- Returns the created assignment

**Gebruik:**

```php src/Domain/Assignments/Actions/CreateAssignmentAction.php
use App\Domain\Assignments\Actions\CreateAssignmentAction;
use App\Domain\Assignments\DTOs\CreateAssignmentDTO;

$action = app(CreateAssignmentAction::class);

$dto = new CreateAssignmentDTO(
    title: 'Review pull request',
    description: 'Review and approve the authentication PR',
    deadline: now()->addDays(2),
    priority: 'high',
    assignee_id: 42
);

$assignment = $action->execute($dto);
```
