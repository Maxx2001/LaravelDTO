## How It Works

<Steps>
  <Step title="Initialize">
    Run `npx ai-docs-kit init` in your Laravel project to copy templates and create config.
  </Step>
  <Step title="Configure">
    Edit `ai-docs.config.json` with your project name, language, and domain path.
  </Step>
  <Step title="Generate">
    Open Cursor, load context, and run the master orchestrator prompt.
  </Step>
  <Step title="Preview">
    Run `mint dev` to preview your generated documentation locally.
  </Step>
</Steps>
